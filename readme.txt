Facebook Like social button for sharing your content with friends to Facebook network.

== Description ==

This plugin enables Facebook Like Button for homepage, pages and posts in different position of layout and also as widget. Spread your website content to Facebook Social Network. Also you can use it as widget or shortcode.

= Plugin Settings =

* Display Facebook Like button before the pages/posts content.
* Display Facebook Like button before the pages/posts content.
* Display Facebook Like Button on the Homepage
* Change Facebook Like button layout (Standard button, Button count or Box count)
* Change Facebook Like button width
* Change Facebook Like button height
* Change Facebook Like button label (Like or Recommend)
* Change Facebook Like button font (Arial, Lucida Grande, Segoe UI, Tahoma, Trebuchet MS, Verdana)
* Change Facebook Like button color scheme (Light or Dark)
* Show faces or not (show facebook profiles below button)
* Preview modifications in real-time

== Installation ==

= How to install =
1. Install automatically through the 'Plugins > Add New' in WordPress Admin, or extract zip file and upload the `facebook-like-button-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= How to use =
- Change settings and manage plugin from 'Settings > Facebook Like Button'. Here you customize the button and choose where you want to display it (Before or after content of pages or/and posts).
- Drag to your sidebar or other location the *Like Button* from 'Appearance > Widgets'. 
- Insert shortcode `[facebooklike]` in the content of any pages or posts.

== Frequently Asked Questions ==

= May I use Facebook Like Button inside of pages or posts content ? = 

Yes. You have to use the shortcode `[facebooklike]`

= Is there a way to turn off the like button for certain pages? =

No. This is not possible in the curent version but will be available in the next release.
